    # StarEvaluate 评星

    # 基础示例：

    <lee-star-evaluate></lee-star-evaluate>
    显示代码

    # 颜色改变：

    <lee-star-evaluate @change="handleChange" initColor="#eee" hoverColor="#FF4500" ></lee-star-evaluate>
    显示代码

    # 星星数量改变：

    <lee-star-evaluate :num=6  @change="handleChange" initColor="#eee" hoverColor="#FF4500" ></lee-star-evaluate>
    显示代码

    # 星星类型改变：

    <lee-star-evaluate type="face" @change="handleChange" initColor="#eee" hoverColor="#FF4500" ></lee-star-evaluate>
    显示代码

    # 初始星星选中：

    <lee-star-evaluate v-model="num"></lee-star-evaluate>

    <script>
    export default {
        data() {
         return {
            num: 1
        }
       },
        methods:{
           handleChange(value) {
            console.log(value);
           }
        }
    }
</script>
    显示代码

    # Attribute

    Parameter Description Type Optional Value Default Value
    v-model 绑定值 Number 1,2,3,4... --
    type 评价图标 String love,star,face love
    num 星星数量 Number 1,2,3,4... 5
    initColor 星星初始颜色 String -- #ccc
    hoverColor 星星划过颜色 String -- #46bd87

    # Event

    Event name Description Callback parameter
    change 返回星星选择几星 value
