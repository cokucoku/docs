    # 属性

    参数 说明 类型 可选值 默认值
    evenlist 配置显示内容 Array content:内容;
    timestamp:时间;
    color:颜色;
    size:点大小;    		 --
    reverse 是否反序 Boolean true,false false
    theme 展示风格 Number 1,2,3,4,5 1
    v-model 初始展示数量 Number - -
    animate 节点动画效果 Boolean animate,空 空
