    # GesturePassword 手势密码

    # 示例：(密码N字形)

    <lee-gesture-password :password="password" @check="check">
</lee-gesture-password> 

    <script>
    export default {
        data() {
              return {
                password:[1,4,7,5,3,6,9],//密码N字形
              }
          },    
          methods:{
            check: function(result) {
                console.log(result)
                //result为err是密码错误，result为success是密码正确
            }                       
          }
    }
</script>
    显示代码

    # GesturePassword 属性

    参数 说明 类型 可选值 默认值
    type 类型 String square,circle,leaf circle
    width 宽度 Number -- 350
    height 高度 Number -- 350
    password 要设置的密码 Array -- --
    successcolor 手势密码颜色 String -- '#67c23a'
    errorcolor 错误时候颜色 String -- '#f56c6c'
    pathwidth 轨迹线粗细 Number -- 8

    # 事件

    事件名称 说明 回调参数
    check 校正事件 返回err或success
