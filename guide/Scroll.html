    # Scroll 滚动条

    # 基础示例：

    在刚刚过去的双11购物节中，京东商城首日战报取得了不俗的成绩，首日开售售出商品5.5亿件，中小型商家成交量较去年有明显上涨，但相比起有直播电商加持的阿里、抖音，成绩略微逊色一些。京东一直是传统电商里步子迈得最小，发展最稳健的平台，在拼多多崛起之时，由于小商品本就不属于京东的核心战略市场，并未受到太多影响，其发展的势头之猛，确实超出所有人的预料。

    京东集团CEO刘强东在本次裁员决策中提到了十分重要的问题，他认为高管们在谈及发展计划时没有将重心放在成本、效率和体验上，空谈故事，这样显然不能为集团带来更好的发展。事实上，自拼多多百亿补贴涉及数码领域时，就已经触及到了京东的核心利益，尤其是苹果全系产品的加入，让两家第三方平台形成正面竞争，但价格往往决定了消费者们最后的选择。

    尝试反击的京东，决定站在同个领域上直面新兴电商的挑战。2019年正式上线的京喜拼拼，与拼多多一样，主打下沉市场，更多的拼购玩法、社区型购物和折扣活动，使其在运营早期取得了不错的成效。但拼多多占领的市场份额太多，不管是京东还是阿里，几乎都无法动摇拼多多已斩获的粉丝群体，京喜拼拼在今年6月开始了大面积的撤店。某种意义上说，京东彻底失去了这部分市场。

    # scrollSelect 属性

    参数 说明 类型 可选值 默认值
    max-height 滚动条区域高度 Number -- 300
    width 滚动条宽度 Number -- 12
    trackColor 滚动条轨道颜色 String -- '#eee'
    barColor 滚动条颜色 String -- '#46bd87'
    layout 配置显示 String bar,arrow bar
