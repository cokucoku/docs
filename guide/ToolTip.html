    # Tooltip 文字提示

    常用于展示鼠标 hover 时的提示信息。

    # 基础示例：

    上边提示文字
    左边提示文字
    右边提示文字
    下边提示文字
    <lee-tool-tip color="#f60" content="上边提示文字" position="top">
   <lee-button>上边</lee-button>
</lee-tool-tip>
    显示代码

    # Tooltip 属性

    参数 说明 类型 可选值 默认值
    color Tooltip颜色 String -- --
    content Tooltip内容 String -- --
    position Tooltip位置 String top,left,right,bottom top
