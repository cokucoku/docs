    # Slide属性

    参数 说明 类型 可选值 默认值
    theme 风格 String red,yellow,black,green,blue blue
    size 尺寸 String s,m,l m
    v-model 初始值 Number,Array -- 0
    show-tip 提示文字 Boolean false,true true
    step 滑动步伐 Number -- 1
    show-stop 停顿点 Boolean false,true false
