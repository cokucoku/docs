    # ScrollSelect 滚动选择

    # 示例：

    <lee-scroll-select type="date" v-model="kg" @change="change"></lee-scroll-select>
<lee-button type="success" @click="show">日期选择</lee-button> 

    <script>
    export default {
        data() {
              return {
               kg:false                
              }
          },    
          methods:{
            show(){
            this.kg=true
            },
            change(data){
             console.log(data)
            }                       
          }
    }
</script>
    显示代码

    # scrollSelect 属性

    参数 说明 类型 可选值 默认值
    type 选择的类型 String date,time date
    color 选择的颜色 String -- '#46bd87'

    # 事件

    事件名称 说明 回调参数
    change 选择后回调 data
